# stonks_bot
virtual paper trading brokerage in discord

Hosted on [repl.it](https://www.repl.it) for free!

# Dependencies

**IF USING REPL.IT, NO NEED TO INSTALL THESE LIBRARY'S**

`pip install discord`

`pip install firebase`

`pip install yfinance`

`pip install pycrypto`
